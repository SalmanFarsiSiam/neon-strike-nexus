How To Setup

Follow these simple steps to get started with MPA Launcher and join the IGI 2 multiplayer community.

1

Download the MPA Launcher

Download the latest version of the MPA Launcher from the Downloads page.

2

Install the Launcher

Run the installer and follow the on-screen instructions. Choose your installation directory and wait for the installation to complete.

3

Launch the Application

Open MPA Launcher from your desktop or start menu. You may need to run as administrator for the first time.

4

Configure Your Settings

Set your in-game name, graphics settings, and other preferences from the launcher settings.

5

Join a Server

Select a server from the server browser and click Join to start playing!

Need Help?

If you're experiencing any issues with the setup process, check our FAQ page or join our Discord community for real-time support.
